# brew-miniapp ☕

一个基于微信小程序的咖啡点单 Demo，包含菜单浏览、规格选择、购物车与订单管理的完整下单闭环。

## 功能

- **菜单点单**：左侧分类侧栏 + 右侧商品分组列表，滚动联动高亮当前分类
- **规格选择**：点击商品弹出规格弹窗，支持杯型（中杯/大杯）、温度（冰/热）、糖度（无糖/少糖/正常/多糖）
- **购物车**：商品增减、删除、清空，实时计算合计金额，一键下单
- **我的订单**：展示历史订单列表，含下单时间、商品明细与规格

## 技术说明

- 原生微信小程序（WXML / WXSS / JS），三 Tab 结构：菜单 / 购物车 / 订单
- 数据层 `miniprogram/utils/store.js` 统一封装，当前基于 `wx.setStorageSync` 本地存储实现
- store 对外暴露抽象接口（菜单查询、购物车增删改、订单创建等），后续可平滑替换为云开发 / 后端 API，业务页面无需改动

## 目录结构

```
miniprogram/
├── app.json              # 全局配置 + tabBar
├── pages/
│   ├── index/            # 菜单点单页
│   ├── cart/             # 购物车页
│   └── orders/           # 我的订单页
└── utils/
    └── store.js          # 数据层（localStorage，预留后端接口）
```

## 本地运行

1. 使用微信开发者工具打开本项目根目录
2. 填入自己的小程序 AppID（或选择测试号）
3. 编译预览即可
